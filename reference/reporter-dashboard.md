# Trace Platform Dashboard Reporter

Produces a modern, multi-section HTML dashboard showing all platform
module scores, violation breakdowns, architecture visualization,
dependency graphs, and historical trend stubs. Designed for enterprise
teams and research organizations.

## Details

Extends the existing
[`reporter_html()`](https://ronaldronnie.github.io/Rtrace/reference/reporter_html.md)
(single-module, diagnostics only) to a full platform health view.
Self-contained — no external CSS/JS dependencies.
