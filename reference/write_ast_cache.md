# Write a project's AST cache to disk

Write a project's AST cache to disk

## Usage

``` r
write_ast_cache(root, cache)
```

## Arguments

- root:

  Character scalar project root.

- cache:

  A named list as returned by
  [`read_ast_cache()`](https://ronaldronnie.github.io/Rtrace/reference/read_ast_cache.md).

## Value

Invisibly, `TRUE`.
