# Changelog

RTrace follows the R community convention of keeping its changelog in
[`NEWS.md`](https://ronaldronnie.github.io/Rtrace/NEWS.md) (this is what
`pkgdown` and CRAN tooling expect). This file exists only so the
changelog is discoverable under its more common cross-language name; see
[NEWS.md](https://ronaldronnie.github.io/Rtrace/NEWS.md) for the actual
entries.
